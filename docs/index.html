Universal Image Converter

A powerful desktop application for converting 50+ image and metadata formats.

python tkinter imagemagick batch-processing exif heic raw-converter

Features

  • Massive Format Support: Converts standard formats (JPG, PNG), Camera RAW, Apple HEIC, HDR, and Medical imaging formats.
  • Batch Processing: Seamlessly convert hundreds of images at once by selecting a full directory.
  • Dynamic JSON Database: Extension capabilities (Alpha, Animation, EXIF, Read/Write support) are driven entirely by a configurable `format_database.json`.
  • Responsive UI: Built with Python/Tkinter, featuring background threading so the app never freezes during processing.

Installation

You need Python 3 and ImageMagick installed on your system.

1. Install ImageMagick

Ubuntu/Debian:

sudo apt-get update && sudo apt-get install imagemagick

macOS:

brew install imagemagick

2. Run the App

python image_converter.py
